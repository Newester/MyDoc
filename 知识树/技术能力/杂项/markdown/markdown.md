**一、字体加强** \
\*\*加粗** \
\_斜体_ \
\*斜体* \
\*\*\*斜体加粗***  
\~~删除线~~ \
\<u>下划线\</u> 

**二、多级标题** \
\# 1 \
\## 2 \
\### 3 \
\#### 4 \
\##### 5 \
\###### 6 

**三、链接** \
图片链接： ![图片内容可不写]（图片文件地址或URL "图片标题" \
自动链接： \<https://www.baidu.com> \
链接： \[链接文本](链接地址) \

**四、分割线** \
\--- 或 *** 或 ___ 行内不含其他内容，三个符号中间可存在空格

**五、代码**  
\```@c \
int main(int argc, char* argv[]) { \
printf("hello world!\n"); \
}    
\```
